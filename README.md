### KeyTao

KeyTao is my main project, involving PCB and case design, as well as firmware coding.
Current features:

* Usage of FreeRTOS kernel.
* N-Key Rollover (NKRO).
* Store macros per key (up to 4 keys).
* 6 LED lightning modes.
